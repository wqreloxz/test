1. Квадратные корни

Арифметический квадратный корень

Арифметическим квадратным корнем из числа a называется неотрицательное число, квадрат которого равен a.

Формула: y = √x. Свойства: √(a * b) = √a * √b, где a ≥ 0, b ≥ 0.


Задание: Вычислите √144 + √25.

2. Квадратные уравнения и Дискриминант

Решение через дискриминант

Стандартный вид: ax² + bx + c = 0 (a ≠ 0).

Дискриминант: D = b² - 4ac

  • D > 0: Два корня: x₁,₂ = (-b ± √D) / 2a
  • D = 0: Один корень: x = -b / 2a
  • D < 0: Действительных корней нет.

Задание: Найдите корни уравнения 2x² - 5x + 3 = 0.

3. Теорема Виета

Теорема Виета для приведенного уравнения

Уравнение вида: x² + px + q = 0.

Сумма корней: x₁ + x₂ = -p

Произведение корней: x₁ * x₂ = q


Задание: Найдите сумму и произведение корней уравнения x² - 7x + 10 = 0.

1. Системы счисления

Двоичная система счисления

Компьютеры используют двоичную систему (биты 0 и 1). Для перевода из десятичной в двоичную нужно делить число на 2 и записывать остатки.

Пример: 5 в десятичной = 101 в двоичной.


Задание: Переведите число 13 в двоичную систему.

2. Алгебра логики

Базовые логические операции

1. Конъюнкция (И, AND): Истинна только если оба высказывания истинны (1 * 1 = 1).

2. Дизъюнкция (ИЛИ, OR): Истинна, если хотя бы одно высказывание истинно (1 + 0 = 1).

3. Инверсия (НЕ, NOT): Меняет значение на противоположное (НЕ 1 = 0).


Задание: Вычислите (1 ИЛИ 0) И (НЕ 0).

3. Основы алгоритмизации (Python)

Условный оператор if-else

Позволяет выполнять разные блоки кода в зависимости от условия.

x = 10 if x > 0: print("Положительное") else: print("Отрицательное или ноль")

Задание: Напишите код на Python, который проверяет, четное ли число a.
